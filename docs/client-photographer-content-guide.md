# Guia para Cliente o Fotografo

Esta guia explica como preparar y compartir el contenido del evento para que podamos mostrarlo dentro de la invitacion digital.

## Que contenido se puede usar

Podemos mostrar:

1. fotos
2. videos
3. contenido previo del evento
4. contenido oficial del evento

Ese contenido se va a mostrar en las secciones:

1. `Ver Fotos Previas`
2. `Contenido Oficial`

## Como preparar la carpeta

1. Crear una carpeta de Google Drive para el evento.
2. Subir ahi el contenido que quieran mostrar.
3. Compartir esa carpeta con la cuenta de Google que les indique Eventechy.

Importante:

1. Compartir la carpeta correcta del evento.
2. No compartir un archivo suelto.
3. No compartir una carpeta de otro evento.

## Recomendacion de estructura

Opcion simple recomendada:

1. una sola carpeta para todo el contenido oficial del evento

Ejemplo:

1. `Contenido oficial`

Si quieren ordenar mejor el material, pueden usar una carpeta con contenido final listo para mostrar y evitar subcarpetas innecesarias.

## Que necesitamos que nos pasen

1. link de la carpeta de Google Drive
2. confirmacion de que la carpeta ya fue compartida con la cuenta indicada
3. nombre del responsable del contenido
4. si habra:
   - solo fotos
   - fotos y videos
5. fecha estimada en la que el contenido estara completo

## Formatos recomendados

### Fotos

Preferimos:

1. JPG
2. PNG
3. WEBP

### Videos

Preferimos:

1. MP4

## Recomendaciones para que cargue mejor

1. usar archivos finales y no duplicados
2. evitar videos excesivamente pesados
3. usar nombres de archivo simples
4. no mover ni borrar la carpeta una vez compartida sin avisarnos
5. si agregan contenido nuevo, subirlo dentro de la misma carpeta ya configurada

## Si hay contenido previo y contenido del evento

Podemos trabajar de dos formas:

### Opcion recomendada

1. usar una sola carpeta para todo

Esto simplifica la configuracion y evita errores.

### Opcion alternativa

1. usar una carpeta para contenido previo
2. usar otra carpeta para contenido oficial del evento

Solo usar esta opcion si realmente necesitan separarlo.

## Buenas practicas

1. subir el contenido con anticipacion
2. confirmar que la carpeta compartida sea la definitiva
3. avisar si van a seguir subiendo material durante el evento
4. evitar cambiar permisos o renombrar la carpeta sin avisar

## Checklist rapido antes de enviar

1. la carpeta contiene el material correcto
2. la carpeta ya fue compartida con Eventechy
3. el link enviado corresponde a la carpeta, no a un archivo
4. las fotos y videos estan listos para mostrarse
5. el contenido pertenece al evento correcto

## Importante

El contenido oficial no pasa por moderacion dentro de la plataforma.

Por eso, el material que suban en esa carpeta es el que quedara disponible para mostrarse en la invitacion digital.
